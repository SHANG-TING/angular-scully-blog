HI, 我是 Neil Xie，有 5 年多關於 Angular、.Net Core、Azure Functions (Serverless)、部分 Java Spring Boot 開發經驗的全端工程師，工作內容大多是開發為企業級通用的後台系統。

我有一個勇於接受新挑戰且不畏懼失敗的心，也有一定的溝通能力，這使得我在工作上如同游擊兵一樣，可以很快速的支援不同部門或團隊，短時間掌握他們熟識的客戶需求、開發語言、框架、工具、習慣、和同事一起討論更好的方案，並漂亮的完成客戶想要的東西。

由於這樣的緣故，使得我接觸過非常多不同領域的語言、框架、設計模式、領域知識，所以很多同事都會在線上、線下找我討論很多不同面向的問題，我也會不吝嗇分享自己的見解，因此我對於自己的工作有著極大的熱情和一定程度的信心，這可以讓我一直不停驅動好奇心，藉此學習更多有幫助的知識和技巧，希望能持續貢獻自己的一切所學給公司、同事、朋友。

前端主要使用 Nx Workspace 建立 Angular Monorepo 專案進行開發，協助配置 ESLint、Prettier、Husky、Tailwind CSS 等設定，若負責專案有獨立開發 UI Library 的話，則會建立 Storybook 提供其他夥伴關於 UI Component 的使用範例和說明，另外開發上有大量使用 RX、NgRx(類似：Redux)，以及撰寫單元或整合測試的經驗，並以提供友善的 UI 帶來良好的使用經驗為目標，此外包含 RWD 設計、跨瀏覽器相容設計、PWA 等相關技術。

後端主要是理解客戶需求，並設計高度正規化的 DB Table、開發大量 Web API (去年在醫療系統專案有翻寫約 500 多隻 API 的經驗)、設計快取機制、設計 Log 機制、開發排程程式去整合第三方平台的資料、網站壓力測試的部分，處理資安相關的問題。
